About the Bootcamp

Welcome to Addis Tech Bootcamp — a program built for the next generation of Ethiopian developers. We offer hands-on training in web design, UI development, and UX principles.

Why join us? Because real skills beat certificates alone. Forget boring textbooks. We learn by building real projects.

Our students have built H2O conservation apps, AI2 powered dashboards, and full e-commerce sites — all from zero prior experience.

Tech Glossary

Front-End
The visual part of a website that users interact with directly in the browser.
Back-End
The server-side logic, databases, and APIs that power the front-end.
Full-Stack
A developer who works on both front-end and back-end systems.
API
Application Programming Interface — a bridge between software systems.

Sample HTML Code You Will Write

Press Ctrl + Shift + I to inspect any page:

<!DOCTYPE html>
<html>
  <head>
    <title>Your Page Title</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Even long file paths like C:\Users\student\projects\mywebsite\index.html are easy to manage once you know the structure.
